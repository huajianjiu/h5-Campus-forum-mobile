# 论坛APP

基于 vue-cli4.0 + webpack 4 + vant ui + sass+ rem 适配方案+axios 封装

### Node 版本要求

`Vue CLI` 需要 Node.js 8.9 或更高版本 (推荐 8.11.0+)。

本示例 Node.js 16

### 启动项目

```bash

npm install

npm run serve
```
